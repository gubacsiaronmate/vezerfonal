# Vezérfonal

## 1. Áttekintés
### Mit csinál ez az app?
Appunk fő célja az, hogy a szervezeteken és iskolákon belüli kommunikációt fejlessze és finomhangolja. Az adminisztrátorok üzeneteket küldhetnek csoportoknak és egyéneknek, akik aztán reagálhatnak az üzenetekre a feladó által kiválasztott emojik valamelyikével.

### Kiknek szól ez az app?

- Cégek alkalmazottjainak
- Iskolai személyzetnek, tanároknak, rendszergazdáknak
- Akárkinek, aki cégen belül üzeneteket kap

### Előnyök

- Tiszta, céltudatos kommunikáció
- Könnyű csoportalapú üzenetküldés
- Gyors visszajelzés küldése és fogadása az emoji reakciók segítségével
- Nincs irreleváns üzenetek által okozott rendetlenség

## 2. Használat kezdése

### Alkalmazás letöltése
1. Töltsd le az alkalmazást az App Store-ról vagy Play Áruházból.
2. Nyisd meg az alkalmazást.

### Bejelentkezés
1. Add meg a cég által biztosított regisztrációs kódot.
2. Adj meg egy érvényes e-mail címet és egy jelszót.
3. Adj meg egy profilképet.
